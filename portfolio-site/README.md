# Clickable e‑portfolio (static)

This folder contains a single-page e‑portfolio website created from the provided CV content (no wording changes; just layout + clickable links).

## Preview locally

Open `portfolio-site/index.html` in your browser.

## Deploy on Vercel

### Option A (easiest): Import from GitHub

1. Push this `portfolio-site/` folder to a GitHub repo.
2. In Vercel, click **Add New → Project** and import the repo.
3. In **Project Settings → Root Directory**, set it to `portfolio-site`.
4. Deploy.

### Option B: Vercel CLI

From the `portfolio-site` directory:

```bash
vercel
```

When prompted, set **Root Directory** to `./` (since you’re already inside `portfolio-site`).
